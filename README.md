## My First Game Mobile - SPACE WAR
 
I worked on this project for 4 months, in this períod i had to do some breaks for learn more about Unity and Game Design to improve the game quality.

I utilized optimizations technics like "object pooling" and " Command Pattern" to improve performance, I heard also many feedbacks about possibles improvements and errors, with them i was capable to improve a lot the gameplay and bug fixes.  

About the Game Design, I utilized technics to improve the player's feeling during the game play, introducing the element of surprise through power-ups on "floating disks" rather than just randomly instantiating them. I also chose to change the enemy's spawn rate
as well as its speed and rate of fire, so that the player is able to feel an evolution every time the boss is defeated.

Exist many improvement to be implemented, besides i decided to pause the project to continue and refine my studies inn Unity and also in Game Design.

## Meu primeiro Game Mobile - SPACE WAR

Trabalhei nesse projeto por cerca de 4 meses, nesse período tive que fazer algumas pausas para aprender mais sobre a Unity e Game Design para melhorar a qualidade do jogo.

Utilizei técnicas de otimização como o "object pooling" e "Command Pattern" para melhorar o desempenho, também ouvi vários feedBacks sobre possíveis melhorias e erros encontrados, com esses feedbacks fui capaz de melhorar muito a jogabilidade e corrigi vários Bugs.

Sobre o Game Design utilizei técnicas referentes a melhorar o sentimento do player enquanto joga, introduzi o elemento da surpresa através dos power Ups em "discos flutuantes" que se movem pelo mapa ao invés de apenas spawná-los aleatoriamente,  também optei por alterar a quantidade de inimigos que aparecem assim como a velocidade e a taxa de disparo para que o player sinta uma evolução cada vez que derrotar o boss tornando o gameplay mais agradável.

Existem várias melhorias a serem implementadas, entretanto decidi pausar o projeto para continuar e aperfeiçoar meus estudos na Unity e também no Game Design.
